# Auth Barebones
A MEAN stack template with authentication

## extra depedencies
```bash
sudo npm install -g nodemon
sudo npm install -g @angular/cli
```

## dependencies of project itself
```bash
cd <project_root>
npm install
```

## run
```bash
cd <project_root>
nodemon
```

## end points
  - /users/authenticate
  - /users/register
  - /users/profile
